## 🛠️ Local Setup

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the bot:**
   ```bash
   python main.py
   ```

## 🚀 Deployment

### Heroku
1. Create new Heroku app
2. Connect GitHub repository
3. Deploy from master branch

1. Connect GitHub repository

cd Bot_handler
